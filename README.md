# Homemade-Yoke-Project
This project focuses on the design, engineering, prototyping, and testing of a custom aerospace-inspired flight yoke system using the gyro feature on a PS5 controller. The project combines mechanical design, electronics, embedded software, and iterative engineering practices to create a functional and realistic control system.

The repository documents the full engineering design process, from initial problem definition and concept development to prototype testing and final implementation.

**Project Goals**
Design a realistic and ergonomic flight yoke
Develop smooth and reliable pitch/roll control mechanisms
Integrate a holder for the PS5 controller for accurate input data
Create modular and maintainable software
Apply real-world engineering documentation practices
Demonstrate iterative prototyping and problem solving
